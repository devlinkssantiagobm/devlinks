## Reglas de negocio


~~~
- Al iniciar o crear un nuevo usuario, sólo se solicita el email y la password. En caso de que se cree uno nuevo usuario, se pedirá una confirmación de usuario


- Al iniciar sesión por primera vez, se pedirá el nombre del usuario. Esto se puede hacer al enviar desde el backend un objeto con la información del usuario y validar si la propiedad username existe 
~~~
